# Mô tả dự án Tiếng Anh cho CV

## Tiếng Anh (English)

**CloudVocab – English Vocabulary Learning App (Self-learning Project)**  
- **Front-end**: Built a responsive single-page UI with ReactJS and CSS, supporting Practice, Flashcard, and Quiz learning modes.  
- **Functions (FE)**: User login/register, personal vocabulary management and bulk import, interactive practice/flashcard/quiz flows with hints, progress statistics, favorites, and word pronunciation.  
- **Back-end**: Designed RESTful APIs with Node.js/Express and MongoDB (User, Vocabulary, UserProgress schemas), applying JWT authentication and bcrypt-based password hashing.  
- **Functions (BE)**: User account management (register/login/profile), per-user vocabulary CRUD and random-word API, fuzzy answer checking using the Levenshtein algorithm, difficulty-based hint generation, and learning statistics aggregation/reset.  
- **Technologies**: ReactJS, JavaScript, Node.js, Express, MongoDB, JWT, bcryptjs, Axios, CSS

## Tiếng Việt

**CloudVocab – Ứng dụng học từ vựng Tiếng Anh (Dự án tự học)**  
- **Front-end**: Xây dựng giao diện web responsive bằng ReactJS và CSS, hỗ trợ các chế độ Luyện tập, Flashcard và Trắc nghiệm.  
- **Chức năng (FE)**: Đăng nhập/đăng ký, quản lý danh sách từ vựng cá nhân và nhập hàng loạt, luồng học tương tác với gợi ý, thống kê tiến độ, đánh dấu yêu thích và nghe phát âm.  
- **Back-end**: Thiết kế RESTful API với Node.js/Express và MongoDB (User, Vocabulary, UserProgress), áp dụng xác thực JWT và mã hóa mật khẩu bằng bcrypt.  
- **Chức năng (BE)**: Quản lý tài khoản (đăng ký/đăng nhập/profile), CRUD từ vựng theo từng user và API lấy từ ngẫu nhiên, kiểm tra đáp án bằng so khớp mờ (Levenshtein), sinh gợi ý theo độ khó và tổng hợp/đặt lại thống kê học tập.  
- **Công nghệ**: ReactJS, JavaScript, Node.js, Express, MongoDB, JWT, bcryptjs, Axios, CSS
